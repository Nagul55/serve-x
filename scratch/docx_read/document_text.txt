TASK 2  
— REMOVE ALL JUNK AND UNUSED CODE
Unused Imports
Remove any import statement where the imported module, component, hook, or function is never referenced in that file.
In Python: remove unused imports flagged by standard linting rules
In JS/TS: remove imports where the identifier never appears in the file body
Unused Variables and Constants
Remove any variable, const, or let declaration that is assigned but never read
Remove any useState hook where neither the state value nor the setter is used
Commented-Out Code Blocks
Remove large blocks of commented-out code. Keep only comments that explain WHY something works. Remove comments that say:
// TODO: fix this later
// old implementation
// testing
// remove this
// not working
// backup code
// kept just in case
Debug Console Logs — Remove All of These
Remove any console.log, console.warn, console.error used for debugging
Logs like "testing", "check this", "debug:", "temp:", "remove later"
Keep only console.error calls that log genuine runtime errors with meaningful messages
Other Junk to Remove
Empty files that contain only a comment header with no actual code
Duplicate utility functions — keep one canonical version in utils/ folder
Dead routes that point to components that don't exist or are never navigated to
Unused CSS class definitions not referenced anywhere in JSX or HTML
Default scaffold content like "Welcome to your new app" or lorem ipsum
Hardcoded test data arrays defined inline inside components if real API data is used
